Getting Started

Installation

# Clone the repository
git clone https://github.com/travis-anderson1982/flowledger.git

# Navigate to project directory
cd flowledger

# Install dependencies
pod install

# Open workspace
open FlowLedger.xcworkspace

Requirements

  • iOS 15.0 or later
  • Xcode 15.0+
  • CocoaPods or Swift Package Manager
  • Plaid API credentials (sign up at Plaid Dashboard)
